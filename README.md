# Selianord MCT Website

Professional business website built using **Blazor WebAssembly (.NET)** for **Selianord MCT — Microsoft Certified Trainer**.

This project showcases a responsive multi-page training and services website designed to present:

- Microsoft certification training services
- software development capabilities
- learning content development expertise
- technical education support offerings

---

## Project Purpose

The goal of this project is to create a modern professional website that demonstrates:

- Microsoft training services
- Azure and developer expertise
- curriculum and learning material development support
- digital solution consulting capabilities

It also serves as a **portfolio project** and **technical showcase** built using Microsoft technologies.

---

## Technologies Used

This solution is built with:

- Blazor WebAssembly (.NET)
- C#
- Razor Components
- HTML5
- CSS3
- Responsive Layout Design
- Visual Studio

---

## Features Implemented

The website currently includes:

### Home Page
Professional landing section introducing Selianord MCT and service offering overview.

### About Page
Professional profile and strengths summary.

### Services Page
Details of available services:

- Corporate Training
- Site Training
- Software Development
- Learning Content Development

### Courses Page
Training areas include:

- Azure Training
- Developer Training
- Data & AI
- Power Platform
- Security Training
- Custom Programmes

### Portfolio Page
Showcases selected professional work areas:

- Microsoft training delivery
- learning content creation
- academic technology support
- business solution development

### Contact Page
Includes contact details and enquiry form layout (UI stage).
