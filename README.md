## Project Purpose

The goal of this project is to create a modern professional website that demonstrates:

- Microsoft training services
- Azure and developer expertise
- curriculum and learning material development support
- digital solution consulting capabilities

It also serves as a **portfolio project** and **technical showcase** built using Microsoft technologies.

---

## Technologies Used

This solution is built with:

- Blazor WebAssembly (.NET)
- C#
- Razor Components
- HTML5
- CSS3
- Responsive Layout Design
- Visual Studio

## Features Implemented

The website currently includes:

### Home Page
Professional landing section introducing Selianord MCT and service offering overview.

- Azure Training
- Developer Training
- Data & AI

Showcases selected professional work areas:

- Microsoft training delivery
- learning content creation
- academic technology support
- business solution development

Includes contact details and enquiry form layout (UI stage).
